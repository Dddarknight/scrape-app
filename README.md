# scrape-app

An app for scraping information from https://scrapeme.live/shop/.

## Instalattion and launch
```
$ git clone git@github.com:Dddarknight/scrape-app.git
$ cd scrape-app
$ pip install poetry
$ make install
$ make build
$ scrape-app
```
